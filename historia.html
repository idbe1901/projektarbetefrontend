Lite historia om Javascript

För att göra historien lite kort och koncist vilket enligt mig gör den lite mer lättöverskådlig så har jag här punktat lite olika viktiga saker ur JavaScripts historia.

  • Mannen som skapade språket JavaScript heter Brendan Eich och det var när han jobbade åt företaget Netscape som han började utveckla språket. År 1995 var det färdigt men har fått utstå mycket krtik på grund av att det inte var väldesignat från början.
  • Från början hette språket Mocha vilket sedan ändrades till LiveScript och tillslut döptes till JavaScript. Språket är även känt under namnet ECMAScript.
  • Språket standardiserades trots all kritik år 1997 under namnet ECMAScript. ECMA är en organisation som som har hand om olika många olika datorstandarder.
  • I början tävlade JavaScript mot Java applets i skapandet av webbsidor men vann ganska fort. Trots likheter i namnet har faktiskt JavaScript och Java inget med varandra att göra.
  • JavaScript har genomgått många revideringar, ES2 (1998), ES3 (1999), ES5 (2009) samt ES6 (2015).
  • Sedan 2015 uppdateras språket årligen vilket har gjort att prestandan är betydligt bättre och JavaScript är ett av de vanligaste språken som används inom programmering.

Det finns givetvis mycket mer att skriva om inom detta ämne men detta får räcka som en kort översikt.
